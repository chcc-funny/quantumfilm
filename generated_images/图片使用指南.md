## 🎨 设计规范总结

### 颜色方案
- ✅ **主色**: 青色系 (#80E5F0 → #1E8A9A)
- ✅ **背景**: 白色/极浅灰 (#FFFFFF → #F8FAFC)
- ✅ **阴影**: 青色半透明 (rgba(30,138,154,0.2))
- ❌ **禁用**: 蓝色系 (#0052FF等)

### 视觉风格
- **Glassmorphism** 玻璃拟态
- 磨砂玻璃纹理
- 柔和渐变
- 精致阴影
- 3D立体感

## 💡 网页开发建议

### 图片优化
```html
<!-- 响应式图片加载示例 -->
<picture>
  <source media="(min-width: 768px)" srcset="01_hero_banner.png">
  <img src="01_hero_banner_mobile.png" alt="量子膜专业汽车贴膜">
</picture>
```

### CSS配合
```css
/* 配合图片的玻璃拟态效果 */
.glass-card {
  background: linear-gradient(135deg,
    rgba(128, 229, 240, 0.1) 0%,
    rgba(30, 138, 154, 0.1) 100%);
  backdrop-filter: blur(10px);
  border: 1px solid rgba(128, 229, 240, 0.3);
  box-shadow: 0 8px 32px rgba(30, 138, 154, 0.2);
}
```

### 图片尺寸对应
- **16:9** → Banner / 场景图 → 建议宽度 1920px
- **4:3** → 流程图 → 建议宽度 1200px
- **1:1** → 图标/产品图 → 建议宽度 800px
